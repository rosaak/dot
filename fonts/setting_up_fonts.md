brew install font-iosevka-aile

###  INSTALLING POWERLINE AND NECESSARY FONTS

```
  pip3 install --user powerline-status
  git clone https://github.com/powerline/fonts
  cd fonts
  ./install.sh
  cd ..
  rm -rf fonts
```
